Check Mongoose Connection Status

Jan 6, 2021

'Mongoose has a readyState property in the Connection route of the API' that is of type property and will return a number representing the current state of the connection, 0-4. These states are as follows:

  • 0 = disconnected
  • 1 = connected
  • 2 = connecting
  • 3 = disconnecting
  • 4 = invalid credentials

These event names will emit when the state changes.
